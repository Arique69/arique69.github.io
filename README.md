# Muhammad Rizq Athariq — Portfolio

Personal portfolio website built with vanilla HTML, CSS, and JavaScript.

🌐 **Live:** [arique69.github.io](https://arique69.github.io)

## Stack

- HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts — Inter + Fira Code

## Sections

- **Hero** — animated typing effect
- **About** — bio and stats
- **Experience** — work history timeline
- **Skills** — tech stack with animated progress bars
- **Projects** — public & private project cards
- **Contact** — email and GitHub links

## Local Development

Just open `index.html` in a browser — no build step needed.
